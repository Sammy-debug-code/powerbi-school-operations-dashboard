# powerbi-school-operations-dashboard-->

This project demonstrates an end-to-end Power BI solution designed to help school management monitor attendance, academic performance, teacher workload, and fee collection using interactive dashboards.

#Business Problems#
Which classes have low attendance trends?|
How are students performing across subjects?|
Are teachers overloaded or underutilized?|
What is the monthly fee collection gap?|

#Data Model#
Implemented a star schema,
Created dimension tables for Students and Classes,
Resolved many-to-many relationships for accurate analysis

#Key KPIs#
Attendance %,
Average Marks,
Fee Collection %,
Teacher Workload (Weekly Hours)

#Tools & Skills Used#

Power BI Desktop,
Power Query,
DAX,
Data Modeling (Star Schema)
